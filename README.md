# Income & Asset Analysis — AskBobAI prototype

A clickable, front-end-only prototype of an **assist-mode** mortgage income and
asset calculation platform for AskBobAI. A user "uploads" borrower documents
(mocked), sees every captured data point with a confidence score and source,
edits any value, runs a calculation with a step-by-step audit lineage, leaves
notes, finalizes, and exports. Income and Asset modules share the same shell and
components.

This is for internal UX validation — no real backend, auth, OCR, or PII.

👉 **See [`RUNNING.md`](./RUNNING.md)** for install/run instructions, a tour of
what to try, and the four mock "seam" files the engineering team plugs into.

```bash
npm install && npm run dev
```
